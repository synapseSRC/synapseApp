## Technical Stack

### Shared Engine (:shared)
- **Networking:** Ktor
- **Backend:** Supabase-kt
- **Local DB:** SQLDelight
- **DI:** Koin

### Android Application (:app)
- **UI:** Jetpack Compose
- **DI:** Hilt
- **Media:** ExoPlayer / Media3

### iOS Application (:iosApp)
- **UI:** SwiftUI
- **Integration:** Consumes shared logic as a static framework.
